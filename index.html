Mis Habilidades

CSS

Hojas de estilo en cascada, es un lenguaje de diseño gráfico para definir y crear la presentación de un documento.

Git

Software de control de versiones pensando en la eficiencia, la confiabilidad y compatibilidad del mantenimiento de aplicaciones.

GitHub

Plataforma web de desarrollo colaborativo de software que permite publicar, almacenar y gestionar proyectos en código abierto usando el sistemad e control de versiones.

GitLab

Plataforma Git y DevOps basada en la nube que brinda ayuda a los desarrolladores a supervisar, probar y desplegar código.

HTML5

HyperText Markup Language, lenguaje de marcado para la elaboración de páginas web.

JavaScript

Lenguaje de programación interpretado. Orientado a objetos, basado en prototipos, imperativo y dinámico.

MongoDB

Base de datos NoSQL de código abierto que almacena datos en documentos BSON, sin tablas relacionales, permitiendo flexibilidad y combinación de múltiples tipos de datos en colecciones.

Nest.js

Framework progresivo de Node.js para la creación de REST APIs escalables, eficientes y seguras del lado del servidor.

Postman

Plataforma que facilita la creación y uso de APIs. La misma permite a los usuarios crear y guardar solicitudes HTTP y HTTPs simples y complejas y realizar documentación de las mismas.

React.js

Librería Javascript de código abierto diseñada para crear interfaces de usuario con el objetivo de facilitar el desarrollo de aplicaciones web.

Tailwind

Framework de CSS para interfaces de usuario modernas y personalizadas, con clases de utilidad aplicables en el código HTML, y flujos de desarrollo para optimizar el peso del CSS.

Docker

Plataforma que utiliza contenedores estandarizados para empaquetar software completo, incluyendo bibliotecas, herramientas y tiempo de ejecución.
